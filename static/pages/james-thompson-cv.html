Compiler Engineer at the Scala Center, Ongoing since 2019, Website

My work focuses on maintenance of open source software surrounding the Scala Programming Language; writing documentation and educational blog posts, and scripting and performing in educational video content. I have also spoken at conferences, assisted user groups, and moderated forums. Below are some notable projects:

2022 - Ongoing

Semantics For TASTy-Query Library, View GitHub Commits

In a greenfield project, my work includes adding types and semantics to TASTy trees (Scala 3’s intermediate representation), developing a semantic API to analyse types and symbols, reading Java and Scala 2 symbols from Java class files.

Oct 2020

Lead for Scala 3’s Enum feature, View Project Milestones, View PRs on Github

I lead a project to stabilise the new enums feature in the Scala 3 compiler. I organised meetings to gather requirements, planned milestones and presented the changes to the Scala Improvement Process committee. The changes were approved and released in Scala 3.0.0.

Sep 2019 -

May 2021

Add Scala 2.13 forward compat with Scala 3, read the accompanying blog post

I lead a project to enable reading Scala 3 library dependencies from the Scala 2 compiler, released in Scala 2.13.4. My blog post teaches users how to leverage this capability to gradually migrate multi-module projects to Scala 3. Maintenance is ongoing. This project exercised test driven development, requiring the creation of a new test framework.

Oct - Nov 2019

Added semanticDB support in Scala 3, Initial PR, View PRs on Github

Added an extractor for semanticDB that analyses typed trees in the Scala 3 compiler, producing semanticDB documents, which record types and positions of symbols, and positions of symbol  references. This enabled critical features for the Metals IDE in Scala 3 projects.

Mar 2021

Let’s Talk About Scala 3 Video Series, Pick any Card, Watch on YouTube

I wrote and performed in a video aimed at teaching beginners how to write data structures in Scala that match the constraints of their problem, starting with the simplest case and gradually increasing complexity, using a card guessing game to motivate the shown examples.

May 2021 /

Jan 2020

Speaker at ScalaCon 2021; f(by) 2020, YouTube (ScalaCon), YouTube (F by)

I gave two versions of a talk for beginners, detailing the Scala 3 transition from Scala 2.13. Attendees were shown how to leverage tools for automated migration, and where to find guides we have produced on migration. I also explained the problem of resolving symbols found in library dependencies, and how Scala 2 and Scala 3 are each able to read dependencies compiled by the other, bidirectionally.
